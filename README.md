# Healthcare-management-analysis-dashboard - Excel
Developed an interactive Healthcare Management Analysis Dashboard to monitor and analyze hospital operations, patient statistics, treatment trends, resource utilization, and financial performance. The project focuses on transforming raw healthcare data into meaningful insights using data analytics and visualization techniques.
Analyzed a healthcare dataset of 55,500 patient records with 21 attributes, covering demographics, medical conditions, hospital details, and billing information using Excel Pivot Tables. 
Identified that Hypertension (13,875 cases) was the most common medical condition, and Female patients (27,750) slightly dominated the dataset. 
Evaluated financial metrics, finding an average billing amount of $25,539, with values ranging from $2,008 to $52,764, highlighting data anomalies and cost variability. 
Performed time-based analysis on admissions (2019–2024), with an average patient age of 51.5 years and average hospital stay of 15.5 days, where Extended Stay (29,505 cases) was the most frequent category. 
Built multiple Pivot Tables to compare admission types, weekdays vs weekends (39,679 weekday admissions), insurance providers, and hospital performance to derive actionable insights.
